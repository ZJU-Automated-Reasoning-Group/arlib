# SRK

Srk (symbolic reasoning kit) is a library for reasoning about logical formulas.

Translated from
https://github.com/zkincaid/srk (OCaml version)

Not all functionalaties are migrated.
 Some components depenc on third-party libraries, e.g., Apron (http://apron.cri.ensmp.fr/library), OCRS (https://github.com/cyphertjohn/OCRS), NTL (http://www.shoup.net/ntl/)
